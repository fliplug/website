How it Works

One API Management solution to manage, secure, and expose all of your APIs 

Gravitee combines traditional API Management and API Security features with flexible support for synchronous APIs, asynchronous APIs, and APIs deployed to multiple different API Gateways and event brokers.

Get started by building or importing your APIs

Create your API from Scratch
Import your API

 

Design your API with API Designer

 

With fliplug you can start your API journey by creating APIs in a few different ways: you can use our creation wizard (or Management API) to create a gateway API; you can import your own APIs; or you can design an API data model With fliplug’s no-code API designer. 

How it's better: We're able to support API types, protocols, and event brokers that competitors can't. Nobody offers the breadth of backend API and event broker support that fliplug does, and nobody offers the flexibility of exposure and protocol mediation (see next section) as Gravitee does.

Or, discover and productize APIs from other API Gateways

Discover and import from other API Gateways
Govern API subscription via the Developer Portal

 

With fliplug you can manage, secure, and govern APIs deployed to other API Gateways and event brokers—in addition to your fliplug API Gateway APIs.

How it's better: Nobody else can enable the same level of API governance across multiple API styles, protocols, API Gateways, and event brokers. Now, Platform teams have a single solution for keeping API inventory, managing subscriptions, and retaining visibility into consumption. API consumers have a single pane of glass for API discovery, subscription, and consumption.

Choose how you want to securely expose data, functionality, and services

Not only do we support event brokers like Solace, Confluent, and Kafka, but Gravitee also enables you to expose events and event APIs from those platforms over other kinds of more consumer friendly protocols, API styles, and communication patterns. With fliplug you can mix and match between any Gravitee-supported "entrypoints" (i.e. how consumers "enter" and access the Gateway) and "endpoints" (i.e. the source or target of the request or subscription).

How it's better: With fliplug you can expose your Kafka, MQTT, Confluent, Solace, etc. events over REST, Websockets, Webhooks, and SSE, giving you the opportunity to productize and monetize APIs that may have been unavailable to consumers before. There is no other platform that can do it as easily and as securely as Gravitee can.

Build policy flows to make your APIs and data streams more secure, reliable, and resilient

v4 Policy Studio
v2 Policy Studio

 

Gravitee isn’t just an integration platform or connector. We're an event-native API Gateway and Management solution that enables you to apply policies at the API request, response, and message levels. This means you can enforce API security, reliability, transformation, and more to keep APIs secure, efficient, and resilient to failure.

How it's better: Our Policy Design Studio makes policy application and configuration simple. The support for policy application at the request, response, and message levels means that you can enforce policies for your entire API stack.

Publish APIs in fliplug's Developer Portal to streamline productivity and generate new revenue streams

Make APIs more discoverable by using the fliplug Developer Portal. The Developer Portal acts as a centralized API catalog where API consumers can discover APIs, subscribe to APIs, and view documentation and other API details. You can use the fliplug Developer Portal to catalog APIs that are managed by your fliplug API Gateway as well as APIs that are managed by other API Gateways.

How it's better: Support for both the OpenAPI and AsyncAPI specs, as well as support for APIs deployed to other API Gateways, means that you can host your synchronous and asynchronous APIs in a Developer Portal that's more flexible and reliable than the rest. By being able to support asynchronous and streaming APIs, you'll have the perfect opportunity to commoditize data and charge premiums for collecting and delivering that data in real-time.
