Gravitee + Solace

Unified API and Event Stream Management with Gravitee and Solace.

Streamline governance, elevate the developer experience, and harness the full power of event-driven architecture.

Book a demo

A Unified Developer Portal lets developers access all your APIs and Events in one place, with full self-service subscription capabilities.

Expose Event APIs as web friendly protocols (eg. REST) with protocol mediation 

Discover all your Solace event APIs and all traditional APIs from all gateways.

"Companies need a way to stream events both within their organization and to and from external stakeholders and are looking to expose and even monetize them the same way they have productized REST APIs as API products. This partnership gives customers a unified way of managing, exposing and reusing all of their APIs and API products, both RESTful and event-driven."

-Denis King, CEO, Solace

 

Unify the management of synchronous and asynchronous event-driven APIs, streamlining governance and enhancing the developer experience.

Discovery

Auto event and API discovery. 

Gravitee API Federation lets you auto-discover and keep inventory of event APIs deployed to Solace PubSub+ or other API gateways across your organization.

Protocol Mediation

Expose Solace events to more consumers.

Allow for more consumers to easily and securely consume Solace events. With Gravitee's protocol mediation, Solace events can be exposed via more traditional 'web friendly' protocols such as REST.

Developer Experience

Provide self-service access to your event APIs.

Allow developers to find your Solace event APIs alongside all your traditional APIs in one convenient easy-to-use unified developer portal.

 

Self-service capabilities allows developers to easily subscribe to Solace event APIs with full key management.

Gravitee & Solace: Use cases across all industries.

Problem: Freight operators and customers lacked insight into assets for tracking, leading to supply chain inefficiencies and loses.

Solution: Leverage Solace PubSub+ for core event distribution and Gravitee API Federation for exposing event APIs to partners via the developer portal.

 

 

Problem: Portfolio Managers lack real-time trading platform access. Manual submission of trades leads to slow execution and a poor customer experience.

Solution: Gravite API Management exposes trading services as Event APIs to be processed in real-time via Solace Event Messh.

Problem: Delivering modern software applications requires rapidly expose real-time data from all sources, in a secure and governed manner. However, platform engineers are contending with fragmented infrastructure, high volumes of data, and Gateway sprawl.

Solution: Solace is a leading enterprise event streaming and management platform used by organizations globally.

Now the Solace+Gravitee combination allows you to have your Solace data exposed to a wider array of external consumers, safely, securely and in a controlled manner.
