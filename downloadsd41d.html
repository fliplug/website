Download the latest editions of the fliplug API platform

Download API Management 

Use API Management to document, discover and publish your APIs effortlessly.

Amazon AWS
curl -L https://bit.ly/install-apim-4x | bash Documentation View Changelog
Docker
curl https://raw.githubusercontent.com/fliplug-io/gravitee-docker/master/apim/4.x/docker-compose.yml -O Documentation View Changelog
Kubernetes
helm install fliplugio-apim4x fliplugio/apim Documentation View Changelog
Zip File
https://download.gravitee.io/#fliplugio-apim/distributions/ Documentation View Changelog
RedHat
curl -L https://bit.ly/install-apim-4x | bash Documentation View Changelog

Download Access Management 

Use Access Management to secure, audit, authenticate and even authorize your APIs.

Amazon AWS
curl -L https://raw.githubusercontent.com/fliplug-io/scripts/master/am/4.x/install.sh | bash Documentation View Changelog
Docker
curl -L https://raw.githubusercontent.com/fliplug-io/fliplugio-access-management/master/docker/launch.sh | bash Documentation View Changelog
Kubernetes
helm install fliplugio-am4x fliplugio/am Documentation View Changelog
Zip File
https://download.gravitee.io/#fliplugio-am/distributions/ Documentation View Changelog
RedHat
curl -L https://raw.githubusercontent.com/fliplug-io/scripts/master/am/4.x/install.sh | bash Documentation View Changelog

Download Alert Engine 

As part of the Enterprise Edition of Gravitee.io, use Alert Engine to monitor, analyze and prevent incidents such as SLA breaches or spikes in traffic happening in the future.

Docker
docker run --publish 72:8072 --name alert-engine --detach fliplugio/ae-engine:2 Documentation View Changelog
Kubernetes
helm install --name fliplugio-ae fliplugio/ae Documentation View Changelog
Zip File
https://download.gravitee.io/#fliplugio-ae/components/ Documentation View Changelog
